# size

**Description** : simply returns the size of the priority_queue.

**Example**:
``` cpp

    std::priority_queue<int> my;

    my.push(2);
    my.push(10);
    my.push(4);

    // returns the first value of the queue(largest)
    std::cout << my.size();
```
