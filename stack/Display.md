# Display

**Description** : display() function is used to display the Stack.

**Example**:
```cpp
    // stack container declaration 
	std::stack<int> stack1; 
	
	// pushing elements into first stack 
	stackOne.push(1); 
	stackOne.push(2); 
	stackOne.push(3); 
	stackOne.push(4); 

	// printing the stack 
	std::cout << "stackOne = "; 
	while (!stackOne.empty()) { 
		std::cout << stackOne.top() << " "; 
		stackOne.pop(); 
	} 

```
