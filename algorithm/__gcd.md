# __gcd

**Example** :

```cpp
#include <iostream> 
#include <algorithm> 

int main() { 
    int number1 = 5;
    int number2 = 255;
	std::cout << "The GCD of the two wanted numbers : " << std::__gcd(number1, number2) << std::endl;
}
```
