# insert

**Description :**
    This method is used to insert elements in std::set.

**Example :**
```cpp
//Run Code To Demonstrate use of set.size()
#include<iostream>
#include<set>

int main(){
    // Create a set object holding integers
    std::set<int> mySet {1,2,3,4,-5};

    // insert element in set
    mySet.insert(7);

    return 0;
}

```
